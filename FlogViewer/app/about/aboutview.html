Notes

Solution Overview
  • Entire viewer is in FlogViewer project (code for module)
  • Front end: Angular 1x using TypeScript
  • Bootswatch Spacelab for styling
  • Kendo UI Core for DateTime pickers
  • Back end: WebAPI controllers with data access using Dapper
"Environments"
  • Put multiple connection strings in your web.config file for logs within different environments
  • The dropdown choice at top right defines the key for the connection string to use
Log Types
  • Aggregate views most important for Usage, Performance, Errors
  • Record listings most important for Errors, Diagnostics
  • Record listing implemented for Error Logs -- others easy to add
Interactivity Notes
  • Exception titles highlighted in yellow
  • Hyperlink on the Id for easy sharing (copy/paste link)
  • Hyperlink the CorrelationId for easy troubleshooting for client-server stuff
